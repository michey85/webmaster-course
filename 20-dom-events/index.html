События в браузере

Типы событий

  • click - нажатие левой клавиши мыши
  • mouseover / mouseout - мыщи наводится на (покидает) элемент
  • keydown / keyup - нажатие / отпускание клавиши клавиатуры
  • focus - установка фокуса на элементе, например, input
  • submit - событие отправки формы
  • ...


Обработка события

  • Через атрибут тега - onevent (например, onclick, onmouseover, onsubmit)
  • Вызов колбэка
  • Выборка ноды и добавление к ней события onevent (по ссылке)
  • Использование "листнеров" addEventListener, removeEventListener


Доступ к содержимому ноды

  • Через контекст this
  • Через объект события
